# NYC Taxi Data Analysis

Análise exploratória de dados de corridas de táxi em Nova York,
com foco em prática de limpeza, exploração e visualização de dados.

# Dados
Os dados utilizados neste projeto *Não estão versionados* por serem grandes. para análise própria faça o Download via Kaggle, o titulo original dos dados é "yellow_tripdata_2016-01".
Alocar os dados renomeando como "dados.csv" em /data.
